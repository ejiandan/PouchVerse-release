# 빠른 시작 가이드 — PouchVerse

> **언어:** [English](Quick-Start.md) | [简体中文](Quick-Start.zh-Hans.md) | [繁體中文](Quick-Start.zh-Hant.md) | [日本語](Quick-Start.ja.md) | 한국어 | [Español](Quick-Start.es.md)

5분 안에 PouchVerse를 시작하세요.

---

## 1단계: 첫 실행

앱을 열면 빈 파일 라이브러리가 준비됩니다. 첫 실행 시, 지원되는 6개 언어로 된 샘플 사용 설명서가 자동으로 추가됩니다.

---

## 2단계: 파일 가져오기

### 방법 A: 다른 앱에서 공유
1. 메일, Safari 등에서 파일을 길게 누릅니다
2. **공유 → PouchVerse로 복사** 탭
3. 파일이 자동으로 라이브러리에 추가됩니다

### 방법 B: 파일 앱에서 가져오기
1. 오른쪽 상단 **⊕** 버튼 탭
2. **파일에서 가져오기** 선택

### 방법 C: LAN 전송으로 수신
→ [전송 가이드](Transfer-Guide.md) 참조

---

## 3단계: 파일 찾기

- 파일 목록을 **아래로 당겨** 검색창 표시 후 키워드 입력
- **필터 아이콘(☰)** 으로 유형, 날짜, 색상, 폴더 등 필터링
- **⇅** 버튼으로 정렬 방식 변경

---

## 4단계: 파일 정리

파일을 **길게 눌러** 부채꼴 메뉴를 엽니다:
- **정리** — 가상 폴더, 태그, 색상 중요도 설정
- **공유** — 시스템 공유 패널로 전송
- **전송** — LAN으로 다른 기기에 전송

---

## 5단계: 비공개 파일

사이드바(☰) → **비공개** → Face ID / Touch ID → 정리 패널에서 **비공개 컬렉션**에 추가.

> 앱이 백그라운드로 전환되면 즉시 **재잠금**됩니다.
