# PouchVerse — Quick Start Guide

> **Language:** English | [简体中文](Quick-Start.zh-Hans.md) | [繁體中文](Quick-Start.zh-Hant.md) | [日本語](Quick-Start.ja.md) | [한국어](Quick-Start.ko.md) | [Español](Quick-Start.es.md)

Get up and running with PouchVerse in 5 minutes.

---

## Step 1: First Launch

Open the app. An empty file library is ready. On first launch, six sample user manuals (one per supported language) are pre-loaded so you always have reference material handy.

---

## Step 2: Import Your Files

There are three ways to bring files into PouchVerse:

### Method A: Share from Another App

1. In any app (Mail, Safari, WeChat, etc.), long-press a file or attachment
2. Tap **Share → Copy to PouchVerse**
3. The file is imported automatically and appears in your library

### Method B: Import from Files App

1. Tap the **⊕** button (top-right)
2. Choose **Import from Files**
3. Browse and select files using the iOS document picker

### Method C: Receive via LAN Transfer

→ See the [Transfer Guide](Transfer-Guide.md) for step-by-step instructions

---

## Step 3: Find Files

- **Pull down** on the file list to reveal the search bar — type to search instantly
- Tap the **filter icon (☰)** to filter by type, date, colour, folder, tag, or usage label
- Tap **⇅** to change sort order (date, size, name, importance)

---

## Step 4: Organise Files

**Long-press** any file to open the radial fan menu:

| Menu Item | Action |
|---|---|
| **Organise** | Set virtual folder, tags, colour importance, usage label |
| **Share** | Send via iOS system share sheet |
| **Transfer** | Send to another device over LAN |

You can also **swipe left** on a file to delete or mark it.

---

## Step 5: Private Files

1. Open the sidebar **(☰)** → tap **Private**
2. Authenticate with Face ID / Touch ID / device passcode
3. In the Organise panel, add any file to **Private Collection**

> The private section **re-locks immediately** whenever the app goes to background.

---

## Tips & Tricks

**Search by Pinyin:** Type `ndbg` to find files named `年度报告` (annual report) — Chinese filenames are automatically indexed by Pinyin initials.

**Audio Transcription:** Open any audio file → tap the `doc.text.magnifyingglass` button to run offline speech-to-text. The result is saved as an SRT file and indexed for search.

**Image Search:** After import, PouchVerse runs on-device OCR on images. Search for text visible in photos (e.g., "invoice", "receipt") or scene labels.

**Duplicate Detection:** Importing a file identical to an existing one (same SHA-256 hash) is silently blocked — storage is shared, no duplicate created.
