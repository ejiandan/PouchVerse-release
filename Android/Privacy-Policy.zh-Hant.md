# 儲物袋 隱私政策

**生效日期：2026 年 5 月 29 日**
**適用產品：儲物袋（PouchVerse）Android 版**
**發布主體：北京藝簡單網絡科技有限公司**

---

## 一、概述

我們堅持「**本地優先 / 離線可用**」的產品原則。儲物袋是一款本地文件聚合管理工具，不要求註冊、不收集帳號、不上傳任何文件或元數據至伺服器。本政策旨在向您透明地說明：本應用在您的 Android 設備上**做什麼**、**不做什麼**，以及您可以**如何控制**。

---

## 二、我們**不**收集的資訊

我們**不會**收集、儲存、上傳或分享下列任何資訊：

- 您的姓名、電話、電郵、身份證號或其他真實身份資訊
- 您的地理位置、設備唯一識別碼（IMEI / Android ID / 廣告識別碼 AAID）或任何追蹤識別碼
- 您匯入、查看、整理、傳輸的任何文件內容或文件名
- 您的搜尋關鍵字、瀏覽記錄、使用行為
- 您的通訊錄、行事曆、相冊元數據、健康數據
- 任何用於追蹤、畫像、行銷、廣告投放的數據

本應用**不內建任何第三方 SDK / 分析 / 廣告 / 推播服務**。

---

## 三、本應用在設備上處理的數據

為實現核心功能，下列數據保存在**您設備的應用私有目錄內**（其他應用無法存取），不會離開您的設備：

| 數據 | 存放位置 | 用途 |
|---|---|---|
| 您匯入的文件本體 | `files/store/` | 提供管理與預覽 |
| 文件元數據（名稱、大小、類型、時間、顏色、用途等） | `databases/pouchverse.db` | 列表、篩選、排序 |
| 全文索引（FTS5） | `databases/pouchverse.db` | 關鍵字搜尋 |
| 縮圖 | `files/thumbs/` | 列表預覽 |
| 雜湊值（SHA-256） | `databases/pouchverse.db` | 內容去重 |
| 偏好設定 | `SharedPreferences` | 影片跳轉步長、語言設定等 |

解除安裝本應用即等同於**完全刪除**上述所有數據。

---

## 四、需要您授權的系統權限

- **儲存 / 文件選擇器**：透過系統文件選擇器匯入文件。
- **媒體**（如使用）：從相冊 / 媒體庫匯入圖片或影片。
- **附近裝置 / 本地網路**：發現裝置並透過區域網路傳輸文件。
- **麥克風**：本應用不主動申請錄音權限；音訊文字提取使用設備本地離線識別，音訊不出本機。
- **生物特徵 / 裝置憑證**：解鎖應用內「私密文件」分區。

本應用**不申請**聯網（含行動數據）、推播通知、相機、定位、通訊錄、行事曆或運動等權限。（注：Android `INTERNET` 權限僅用於區域網路功能的本地 Wi-Fi 通訊端通訊，不與任何遠端伺服器交互。）

---

## 五、網路行為

- **不連接互聯網**：本應用不會向任何遠端伺服器發送或接收數據。
- **區域網路透傳**：僅當您主動使用「區域網路傳輸」功能時，本應用會透過 mDNS / NSD 發現同一區域網路內的對端裝置，並透過自訂 TCP 協議（連接埠 52013）直接傳輸您選擇的文件。
  - 區域網路傳輸**不經過任何中繼伺服器**，發送方與接收方點對點直連。
  - 傳輸不加密，請勿在不可信的公共 Wi-Fi 網路下傳輸敏感數據。

---

## 六、私密文件

「私密文件」功能透過 Android 系統 BiometricPrompt（指紋 / 臉部 / 裝置 PIN / 圖案 / 密碼）提供**應用內的二級存取控制**，用於隱藏私密資料夾中的內容。請注意：

- 私密文件**未在磁碟上加密**；其隱私邊界依賴「成員關係標記 + 記憶體解鎖態 + 介面隱藏」。
- 應用切換至背景或退出後**立即重新鎖定**，重新開啟需再次身份驗證。
- 在 Root 裝置上，或他人在知道裝置憑證的情況下實體取得裝置時，文件可能被讀取。

---

## 七、未成年人保護

本應用面向一般使用者群體，不針對未成年人提供專門服務。如未滿 14 歲的未成年人使用，請在監護人指導下使用，並由監護人協助管理裝置權限與匯入內容。

---

## 八、第三方開源元件

本應用靜態整合了以下開源元件，**不透過它們傳輸任何數據**：

- **FFmpeg**（LGPL 2.1+）：本地影片 / 音訊解碼
- **unrar**（UnRAR License，僅解壓縮）：RAR 解壓縮
- **zlib**（zlib license）：DEFLATE 解壓縮
- **SQLite3**（公有領域）：本地資料庫
- **AndroidX / Jetpack Compose**（Apache 2.0）：UI 框架

詳細授權文本可在 App 內「側邊欄 → 授權」查看。

---

## 九、數據匯出與刪除

- **匯出**：透過列表行長按 → 分享，或直接將文件分享至系統文件管理器 / 其他 App。
- **刪除**：列表行向左滑動刪除，或在系統設定中解除安裝本 App，將徹底移除所有本地數據。
- 由於數據完全位於本機，**我們無法、也不會**遠端復原您已刪除的數據。
