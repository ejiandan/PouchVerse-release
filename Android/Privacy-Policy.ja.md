# PouchVerse プライバシーポリシー

**発効日：2026 年 5 月 29 日**
**対象製品：PouchVerse（Android 版）**
**発行者：北京 EJIANDAN Network Technology Co., Ltd.**

---

## 1. 概要

当社は「**ローカルファースト / オフライン対応**」の製品理念を掲げています。PouchVerse はローカルファイル統合管理ツールであり、アカウント登録不要、個人データの収集なし、ファイルやメタデータをサーバーにアップロードすることは一切ありません。本ポリシーは、本アプリがお客様の Android デバイス上で何を行い、何を行わないか、そしてお客様がどのように管理できるかを透明な形でご説明するものです。

---

## 2. 収集**しない**情報

当社は以下の情報を**一切**収集・保存・アップロード・共有しません：

- お客様の氏名・電話番号・メールアドレス・身分証明書番号その他の個人識別情報
- お客様の位置情報・デバイス固有識別子（IMEI / Android ID / 広告識別子 AAID）・追跡識別子
- お客様がインポート・閲覧・整理・転送したファイルの内容やファイル名
- 検索キーワード・閲覧履歴・アプリ内利用行動
- 連絡先・カレンダー・フォトライブラリのメタデータ・ヘルスデータ
- トラッキング・プロファイリング・マーケティング・広告配信に使用されるデータ

本アプリには**サードパーティの SDK / 分析サービス / 広告ネットワーク / プッシュ通知サービスは一切組み込まれていません**。

---

## 3. デバイス上で処理されるデータ

コア機能を提供するため、以下のデータは**アプリのプライベートディレクトリ内**（他のアプリからアクセス不可）にのみ保存され、デバイスの外に出ることはありません：

| データ | 保存場所 | 目的 |
|---|---|---|
| インポートしたファイル本体 | `files/store/` | ファイル管理とプレビュー |
| ファイルメタデータ（名前・サイズ・種類・日時・カラータグ等） | `databases/pouchverse.db` | 一覧表示・フィルター・並べ替え |
| 全文検索インデックス（FTS5） | `databases/pouchverse.db` | キーワード検索 |
| サムネイル | `files/thumbs/` | 一覧プレビュー |
| コンテンツハッシュ（SHA-256） | `databases/pouchverse.db` | 重複検出 |
| 設定情報 | `SharedPreferences` | 動画シーク幅・言語設定等 |

本アプリをアンインストールすると、上記のデータはすべて**完全に削除**されます。

## 4. システム権限

- **ストレージ / ドキュメントピッカー**：システムのドキュメントピッカー経由でファイルをインポートします。
- **メディア**（使用する場合）：ギャラリー / メディアライブラリから写真や動画をインポートします。
- **近くのデバイス / ローカルネットワーク**：デバイスを検出し、LAN 経由でファイルを転送します。
- **マイク**：本アプリは録音権限を能動的には要求しません。音声テキスト抽出はデバイス上のオフライン認識を使用し、音声はデバイス外に送信されません。
- **生体認証 / デバイス認証情報**：アプリ内「プライベートファイル」セクションのロック解除に使用します。

本アプリはインターネット接続（モバイルデータ含む）・プッシュ通知・カメラ・位置情報・連絡先・カレンダー・モーションの権限を**要求しません**。（注：Android の `INTERNET` 権限は LAN 機能のローカル Wi-Fi ソケット通信にのみ使用され、外部サーバーとは通信しません。）

---

## 5. ネットワーク動作

- **インターネット非接続**：本アプリは外部サーバーとのデータ送受信を一切行いません。
- **LAN 転送**：「LAN 転送」機能を能動的に使用する場合のみ、本アプリは mDNS / NSD を使用して同一ローカルネットワーク上のデバイスを検出し、カスタム TCP プロトコル（ポート 52013）を介してファイルをピアツーピアで直接転送します。
  - LAN 転送は**中継サーバーを経由しません**；送受信者はポイントツーポイントで直接接続します。
  - 転送は暗号化されていません。信頼できない公共 Wi-Fi ネットワークでの機密ファイルの転送はお控えください。

---

## 6. プライベートファイル

「プライベートファイル」機能は、Android の BiometricPrompt（指紋 / 顔認証 / デバイス PIN / パターン / パスワード）を使用して、非公開フォルダのコンテンツを保護する**アプリ内二次アクセス制御**を提供します。ご注意ください：

- プライベートファイルは**ディスク上では暗号化されていません**。プライバシーの境界は、メンバーシップフラグ・メモリ内のロック解除状態・UI による隠蔽に依存しています。
- アプリがバックグラウンドに移行または終了すると**即座に再ロック**され、次回アクセス時に再認証が必要です。
- Root 化されたデバイス、またはデバイス認証情報を知る者に物理的にアクセスされた場合、ファイルが読み取られる可能性があります。

## 7. 子どものプライバシー

本アプリは一般ユーザーを対象としており、子ども向けのサービスではありません。14 歳未満の未成年者が使用する場合は、保護者の監督のもとでご使用いただき、デバイスの権限とインポートするコンテンツを保護者が管理してください。

---

## 8. サードパーティオープンソースコンポーネント

本アプリは以下のオープンソースライブラリを静的にリンクしています。**いずれを通じてもデータは送信されません。**

- **FFmpeg**（LGPL 2.1+）：ローカル動画 / 音声デコード
- **unrar**（UnRAR ライセンス、解凍のみ）：RAR アーカイブ解凍
- **zlib**（zlib ライセンス）：DEFLATE 伸張
- **SQLite3**（パブリックドメイン）：ローカルデータベース
- **AndroidX / Jetpack Compose**（Apache 2.0）：UI フレームワーク

ライセンスの全文はアプリ内の**サイドバー → ライセンス**でご確認いただけます。

---

## 9. データのエクスポートと削除

- **エクスポート**：ファイル行を長押し → 共有、またはシステムのファイルマネージャー / 他のアプリに直接共有します。
- **削除**：ファイル行を左にスワイプして削除するか、システム設定からアプリをアンインストールすることですべてのローカルデータを完全に削除できます。
- データはすべてデバイス上にのみ存在するため、**削除されたデータをリモートで復元することは当社には不可能であり、行うつもりもありません**。
