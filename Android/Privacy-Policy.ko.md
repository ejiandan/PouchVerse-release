# PouchVerse 개인정보 처리방침

**시행일: 2026년 5월 29일**
**적용 제품: PouchVerse (Android)**
**게시 기관: Beijing EJIANDAN Network Technology Co., Ltd.**

---

## 1. 개요

당사는 **로컬 우선 / 오프라인 사용 가능** 제품 철학을 지향합니다. PouchVerse는 로컬 파일 통합 관리 도구로, 계정 등록이 필요 없고 개인 데이터를 수집하지 않으며 파일이나 메타데이터를 서버에 업로드하지 않습니다. 본 방침은 본 앱이 귀하의 Android 기기에서 무엇을 하고 하지 않는지, 그리고 귀하가 어떻게 제어할 수 있는지를 투명하게 설명합니다.

---

## 2. 수집하지 **않는** 정보

당사는 다음 정보를 **절대로** 수집, 저장, 업로드 또는 공유하지 않습니다:

- 귀하의 이름, 전화번호, 이메일 주소, 주민등록번호 또는 기타 개인식별정보
- 귀하의 위치 정보, 기기 고유 식별자(IMEI / Android ID / 광고 ID AAID) 또는 추적 식별자
- 귀하가 가져오기, 열람, 정리, 전송하는 파일 내용 또는 파일명
- 검색어, 검색 기록 또는 앱 내 사용 행동
- 연락처, 캘린더, 사진 라이브러리 메타데이터 또는 건강 데이터
- 추적, 프로파일링, 마케팅 또는 광고 목적으로 사용되는 데이터

본 앱에는 **서드파티 SDK, 분석 서비스, 광고 네트워크 또는 푸시 알림 서비스가 일절 포함되어 있지 않습니다**.

---

## 3. 기기에서 처리되는 데이터

핵심 기능을 제공하기 위해 다음 데이터는 **앱의 프라이빗 디렉터리**(다른 앱이 접근 불가)에만 저장되며 기기 밖으로 나가지 않습니다:

| 데이터 | 저장 위치 | 목적 |
|---|---|---|
| 가져온 파일 본체 | `files/store/` | 파일 관리 및 미리보기 |
| 파일 메타데이터(이름, 크기, 유형, 날짜, 컬러 태그 등) | `databases/pouchverse.db` | 목록 표시, 필터링, 정렬 |
| 전문 검색 색인(FTS5) | `databases/pouchverse.db` | 키워드 검색 |
| 썸네일 | `files/thumbs/` | 목록 미리보기 |
| 콘텐츠 해시(SHA-256) | `databases/pouchverse.db` | 중복 감지 |
| 환경설정 | `SharedPreferences` | 동영상 탐색 단계, 언어 설정 등 |

PouchVerse를 삭제하면 위의 모든 데이터가 **영구적으로 삭제**됩니다.

## 4. 시스템 권한

- **저장소 / 문서 선택기**: 시스템 문서 선택기를 통해 파일을 가져옵니다.
- **미디어** (사용 시): 갤러리 / 미디어 라이브러리에서 사진이나 동영상을 가져옵니다.
- **근처 기기 / 로컬 네트워크**: LAN을 통해 기기를 검색하고 파일을 전송합니다.
- **마이크**: 본 앱은 녹음 권한을 능동적으로 요청하지 않습니다. 오디오 텍스트 추출은 기기의 오프라인 인식을 사용하며 오디오는 기기 밖으로 나가지 않습니다.
- **생체 인식 / 기기 자격 증명**: 앱 내 '비공개 파일' 섹션 잠금 해제에 사용합니다.

본 앱은 인터넷 접속(모바일 데이터 포함), 푸시 알림, 카메라, 위치, 연락처, 캘린더 또는 동작 권한을 **요청하지 않습니다**. (참고: Android `INTERNET` 권한은 LAN 기능의 로컬 Wi-Fi 소켓 통신에만 사용되며 원격 서버와는 통신하지 않습니다.)

---

## 5. 네트워크 활동

- **인터넷 비연결**: 본 앱은 원격 서버와 데이터를 주고받지 않습니다.
- **LAN 전송**: 'LAN 전송' 기능을 능동적으로 사용하는 경우에만 본 앱은 mDNS / NSD를 사용하여 동일 로컬 네트워크의 상대 기기를 검색하고 커스텀 TCP 프로토콜(포트 52013)을 통해 선택한 파일을 직접 P2P 방식으로 전송합니다.
  - LAN 전송은 **중계 서버를 거치지 않습니다**. 발신자와 수신자는 직접 P2P 연결됩니다.
  - 전송은 암호화되지 않습니다. 신뢰할 수 없는 공용 Wi-Fi 네트워크에서 민감한 파일을 전송하지 마십시오.

---

## 6. 비공개 파일

'비공개 파일' 기능은 Android BiometricPrompt(지문 / 얼굴 인식 / 기기 PIN / 패턴 / 비밀번호)를 사용하여 비공개 폴더의 내용을 보호하는 **앱 내 2차 접근 제어**를 제공합니다. 다음 사항에 유의하십시오:

- 비공개 파일은 **디스크에서 암호화되지 않습니다**. 프라이버시 경계는 멤버십 플래그, 메모리 내 잠금 해제 상태 및 UI 숨김에 의존합니다.
- 앱이 백그라운드로 전환되거나 종료되면 **즉시 재잠금**되며 다음 접근 시 재인증이 필요합니다.
- 루팅된 기기에서 또는 기기 자격 증명을 아는 사람이 기기에 물리적으로 접근한 경우 파일이 읽힐 수 있습니다.

## 7. 아동 개인정보 보호

본 앱은 일반 사용자를 대상으로 하며 아동을 위한 서비스가 아닙니다. 만 14세 미만의 미성년자가 사용하는 경우 보호자의 감독하에 사용하고, 기기 권한 및 가져오는 콘텐츠는 보호자가 관리해야 합니다.

---

## 8. 서드파티 오픈소스 컴포넌트

본 앱은 다음 오픈소스 라이브러리를 정적으로 링크합니다. **이를 통해 데이터가 전송되지 않습니다.**

- **FFmpeg** (LGPL 2.1+): 로컬 동영상 / 오디오 디코딩
- **unrar** (UnRAR 라이선스, 압축 해제 전용): RAR 아카이브 압축 해제
- **zlib** (zlib 라이선스): DEFLATE 압축 해제
- **SQLite3** (퍼블릭 도메인): 로컬 데이터베이스
- **AndroidX / Jetpack Compose** (Apache 2.0): UI 프레임워크

전체 라이선스 텍스트는 앱 내 **사이드바 → 라이선스**에서 확인할 수 있습니다.

---

## 9. 데이터 내보내기 및 삭제

- **내보내기**: 파일 행을 길게 누른 후 → 공유, 또는 시스템 파일 관리자 / 다른 앱으로 직접 공유합니다.
- **삭제**: 파일 행을 왼쪽으로 스와이프하여 삭제하거나, 시스템 설정에서 앱을 삭제하면 모든 로컬 데이터가 영구적으로 제거됩니다.
- 모든 데이터는 기기에만 저장되므로 **삭제된 데이터를 원격으로 복구하는 것은 불가능하며 당사도 이를 시도하지 않습니다**.
