# PouchVerse（储物袋）

> **작은 것에 세우고, 간결하게 행한다 · 모든 파일을 하나로, 순간에 열람**  
> 흩어진 파일을 통합 라이브러리에 모아 검색·미리보기·전송을 한 번에.

PouchVerse는 **로컬 우선 / 오프라인 지원** 개인 파일 관리 앱입니다. 계정 등록 불필요, 클라우드 업로드 없음, 구독 없음. 모든 파일은 **기기 내에만** 저장됩니다.

---

## 만문귀일（萬文歸一）

분산된 디지털 파일을 처리하는 통합 시스템.

파일의 혼란은 수량에서 오는 것이 아니라, "한 곳에 강제로 넣어야 하는" 구조에서 옵니다.

"2026년 여름 해변 팀빌딩" 사진은 전통적인 파일 시스템에서 하나의 위치를 선택해야 합니다: 폴더가 그것이 어디에 있는지를 결정합니다——당신이 그것을 어떻게 기억하는지가 아니라.

우리는 그 제약을 제거했습니다. 파일은 더 이상 경로에 묶이지 않고, 정보 자체로 존재합니다. 저장소는 숨겨지고, 구조는 해방되며, 동일한 콘텐츠가 여러 차원에 동시에 속할 수 있습니다.

그것은 동시에 "2026", "여름", "해변", "팀빌딩"에 속할 수 있습니다.

---

파일이 시스템에 진입하는 순간, 이해 가능한 정보 단위로 재구성됩니다:

- OCR 추출
- 전문 색인
- 지문 계산
- 썸네일 생성
- 다국어 검색 향상 (병음 / 첫 글자)

이것은 "파일 정리"가 아닙니다——파일이 처음부터 찾을 수 있도록 하는 것입니다.

---

시스템은 세 가지 계층으로 구성됩니다:

- **스마트폰 / 태블릿** — 디지털 자산의 핵심 진입점 및 관리 허브
- **데스크톱** — 심층 처리 및 동기화 노드
- **TV / 대화면** — 전체 정보의 스트리밍 소비 터미널

파일 시스템을 복제하지 않고, 동일한 정보 구조를 공유합니다.

---

파일은 더 이상 폴더에 속하지 않습니다.\
그것은 당신에게 속하고, 모든 상황에서 흐릅니다.

---

## ✨ 왜 PouchVerse인가요?

스마트폰의 파일은 메일·SNS·브라우저에 흩어져 있습니다. 기존 파일 관리자는 데스크탑의 '폴더 트리' 방식을 그대로 가져와 이런 단편화에 효과적으로 대응하지 못합니다.

**PouchVerse는 새로운 접근 방식을 취합니다:**

- **만문귀일** — 모든 앱에서 파일을 가져와 통합 인덱싱
- **물리적 중복 제거** — SHA-256 해시로 동일 콘텐츠는 하나의 실체로 통합
- **6차원 유연 정리** — 태그·중요도·가상 폴더·용도 레이블로 자유 분류
- **밀리초 오프라인 검색** — 파일명·본문·이미지 OCR·오디오 음성 변환(수동 트리거)
- **전 형식 미리보기** — 80종 이상 지원, 외부 앱 전환 불필요
- **크로스플랫폼 전송** — LAN P2P 고속 전송, Apple 기기 간 Wi-Fi 불필요
- **생체인증 프라이빗 공간** — Face ID / Touch ID 보호, 100% 로컬
