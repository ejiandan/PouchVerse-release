# PouchVerse（储物袋）

> **작은 것에 세우고, 간결하게 행한다 · 모든 파일을 하나로, 순간에 열람**  
> 흩어진 파일을 통합 라이브러리에 모아 검색·미리보기·전송을 한 번에.

PouchVerse는 **로컬 우선 / 오프라인 지원** 개인 파일 관리 앱입니다. 계정 등록 불필요, 클라우드 업로드 없음, 구독 없음. 모든 파일은 **기기 내에만** 저장됩니다.

---

## 만문귀일（萬文歸一）

### 파일은 한 번도 한 곳에 있었던 적이 없다

현실의 파일은 항상 흩어져 있습니다.

스마트폰 갤러리, 채팅 기록, 다운로드 폴더, 각 앱에 각각 하나씩.\
자료를 찾으려면 여러 곳을 왔다 갔다 해야 하고, 심지어 여러 번 중복 저장하게 됩니다.

처음의 아이디어는 사실 간단했습니다:\
모든 파일을 한 곳에 모아 통합 관리한다.

"파일 찾기"가 더 이상 운에 의존하지 않도록.

### 한 장의 사진이 만든 변화

2026년 여름 해변 팀빌딩에서 찍은 사진이 있습니다.

전통적인 방식으로 저장하려면 어딘가의 폴더에 넣어야 합니다:\
"2026"? "팀빌딩"? "해변"? "업무 사진"?

하지만 그 사진 자체는 하나의 위치에만 속하는 것이 아닙니다.

그래서 우리는 "저장"이라는 것을 다시 생각하기 시작했습니다.\
파일은 더 이상 특정 폴더에 속할 필요가 없으며, 처음부터 경로에 묶이지 않습니다.

물리적 저장소는 숨겨지고, 사용자는 더 이상 구조를 직접 마주하지 않습니다.\
대신 정보 자체가 해방되어 여러 차원에서 동시에 정리될 수 있습니다:\
2026, 여름, 해변, 팀빌딩 — 모두 그 사진의 진입점이 될 수 있습니다.

### 파일은 들어오는 순간 이해된다

파일이 시스템에 임포트되는 시점에 이미 변화가 일어납니다.

더 이상 정적인 파일이 아니라, 재처리된 정보 단위가 됩니다:\
지문이 계산되고, 콘텐츠가 인식되고, 텍스트가 추출되고, OCR이 완료되고, 인덱스가 구축되고, 썸네일이 생성되고, 중국어 텍스트는 병음과 첫 글자 처리가 이루어집니다.

이 모든 것이 들어오는 순간에 일어납니다.

전통적인 폴더는 이것을 하지 않습니다.\
폴더는 "넣는 것"만 담당합니다 — 여기서는 "찾을 수 있게 하는 것"을 담당합니다.

파일은 들어오는 순간부터 이미 이해 가능하고, 검색 가능하고, 연관 가능해집니다.

### 정보는 모든 기기 사이를 흐른다

정보가 구조화되면 더 이상 기기에 묶이지 않습니다.

스마트폰과 태블릿이 핵심 노드가 됩니다: 디지털 자산의 임포트·관리·보유 담당.\
데스크톱이 처리 노드가 됩니다: 중요한 콘텐츠를 정리·가공하고 시스템으로 환류시킵니다.\
TV가 소비 노드가 됩니다: 모든 정보를 즉시 동기화하고 스트림으로 탐색·사용할 수 있습니다.

각자 별도의 파일 시스템을 유지하는 것이 아니라, 하나의 정보 구조를 함께 사용합니다.

파일은 더 이상 "어느 기기에 있는 것"이 아니라, 항상 같은 시스템에 속하고 다양한 맥락에서 흐릅니다.

## ✨ 왜 PouchVerse인가요?

스마트폰의 파일은 메일·SNS·브라우저에 흩어져 있습니다. 기존 파일 관리자는 데스크탑의 '폴더 트리' 방식을 그대로 가져와 이런 단편화에 효과적으로 대응하지 못합니다.

**PouchVerse는 새로운 접근 방식을 취합니다:**

- **만문귀일** — 모든 앱에서 파일을 가져와 통합 인덱싱
- **물리적 중복 제거** — SHA-256 해시로 동일 콘텐츠는 하나의 실체로 통합
- **6차원 유연 정리** — 태그·중요도·가상 폴더·용도 레이블로 자유 분류
- **밀리초 오프라인 검색** — 파일명·본문·이미지 OCR·오디오 음성 변환(수동 트리거)
- **전 형식 미리보기** — 80종 이상 지원, 외부 앱 전환 불필요
- **크로스플랫폼 전송** — LAN P2P 고속 전송, Apple 기기 간 Wi-Fi 불필요
- **생체인증 프라이빗 공간** — Face ID / Touch ID 보호, 100% 로컬
