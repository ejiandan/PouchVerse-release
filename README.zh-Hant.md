# 儲物袋 · PouchVerse

> **構於微，行於簡 · 匯集群文，速覽無滯**  
> 將散落各處的文件匯入統一的本地庫，搜索、預覽、互傳一氣呵成。

儲物袋是一款**本地優先 / 離線可用**的個人文件管理應用，專為**手機與平板**（iOS 和 Android）打造；**儲物袋 Transfer**（macOS / Windows）為桌面平台提供高速局域網互傳功能。無需註冊帳號，無雲端上傳，無訂閱。您的全部文件**始終存儲於您的裝置本地**，完全由您掌控。

## 萬文歸一

### 文件從來都不在一個地方

現實裡的文件總是散落的。

手機相簿、聊天記錄、下載目錄、各個應用裡各自一份。
想找一份資料，經常要在不同地方來回切換，甚至重複儲存多次。

最初的想法其實很簡單：
把所有文件收進一個地方，統一管理。

讓「找文件」這件事，不再靠運氣。

### 一張照片引出的變化

有一張照片，是2026年夏天海邊團建留下的。

如果用傳統方式存放，它必須被放進某一個文件夾：
「2026」？「團建」？「海邊」？「工作照片」？

但它本身不只屬於一個位置。

於是我們開始重新思考「存放」這件事。
文件不再必須歸屬某一個文件夾，而是從一開始就不再被路徑綁定。

物理儲存被隱藏起來，用戶不再直接面對結構。
取而代之的是信息本身被釋放出來，可以同時被多個維度組織：
2026、夏天、海邊、團建，都可以成為它的入口。

### 文件在進入的那一刻就被理解

當文件被導入系統時，它就已經發生變化。

它不再只是一個靜態文件，而是被重新處理的信息單元：
指紋被計算，內容被識別，文本被提取，OCR被完成，索引被建立，縮略圖被生成，中文還會進行拼音與首字母處理。

這一切發生在進入的瞬間。

傳統文件夾不會做這些事情。
它只負責「放進去」，而這裡負責「讓它可以被找回來」。

文件從進入開始，就已經變得可理解、可檢索、可關聯。

### 信息在所有設備之間流動

當信息被結構化之後，它不再被設備限制。

手機與平板成為核心節點：負責導入、管理與承載數位資產。
桌面端成為處理節點：用於整理與加工重要內容，再回流系統。
電視端成為消費節點：所有信息可以瞬時同步，以流的方式被瀏覽與使用。

它們不再各自維護一套文件系統，而是共同使用同一套信息結構。

文件不再「在哪個設備裡」，而是始終屬於同一個系統，在不同場景中流動。

## ✨ 為什麼選擇儲物袋？

手機上的文件散落在微信、電郵、瀏覽器……傳統文件管理器照搬桌面「目錄樹」邏輯，無法高效應對這種碎片化。

**儲物袋採用全新方案：**

- **萬文歸一** — 從任意 App 導入，統一收納、建索引、供預覽。
- **物理級去重** — SHA-256 數字指紋精準識別相同內容（與文件名無關），幫助 128 GB / 256 GB 設備用戶節省大量空間。
- **六維柔性組織** — 按您的思維邏輯組織：標籤、重要度、虛擬文件夾、用途標注——一個文件，多重歸屬，無物理複製。
- **毫秒級離線檢索** — 全文檢索覆蓋文件名、文檔正文、圖片 OCR 文本、中文拼音首字母，以及按需觸發的音頻語音轉寫。全部在設備本地完成。
- **全格式速覽** — 文檔、代碼、視頻、音頻、壓縮包、圖片……一鍵直接預覽，無需跳轉外部應用。
- **專業影音播放** — 自研雙區精準視頻跳轉控件，封面信息豐富的音頻播放器。
- **跨平台互傳** — 局域網點對點高速直傳，蘋果設備間無需 WiFi 即可直連。
- **生物識別私密空間** — Face ID / Touch ID 保護的私密文件夾，100% 本地存儲。

## 🛡️ 隱私承諾

儲物袋堅守**本地優先 / 離線可用**的產品原則：

- ❌ 無帳號註冊
- ❌ 不收集任何個人數據
- ❌ 不上傳任何文件到服務器
