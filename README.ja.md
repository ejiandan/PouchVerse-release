# PouchVerse（储物袋）

> **微に構え、簡に行う · すべてのファイルを一つに、瞬時に閲覧**  
> 散らばったファイルを統一ライブラリに集約し、検索・プレビュー・転送をシームレスに。

PouchVerse は**ローカルファースト / オフライン対応**の個人ファイル管理アプリです。アカウント登録不要、クラウドアップロードなし、サブスクなし。すべてのファイルは**お使いのデバイス内のみ**に保存されます。

---

## ✨ なぜ PouchVerse なのか？

スマートフォン上のファイルはメール・SNS・ブラウザに散乱しています。従来のファイルマネージャーは、デスクトップの「フォルダツリー」をそのまま移植したものが多く、このような断片化に対応できません。

**PouchVerse は新しいアプローチを取ります：**

- **万文帰一** — あらゆるアプリからファイルを集約し、統一インデックス管理
- **物理レベル重複排除** — SHA-256 フィンガープリントでコンテンツが同一なら 1 つの実体に統合
- **6 次元フレキシブル整理** — タグ・重要度・仮想フォルダ・用途ラベルで自由分類
- **ミリ秒オフライン検索** — ファイル名・本文・画像 OCR・音声文字起こし（手動トリガー）
- **全形式プレビュー** — 80 種以上対応。外部アプリへの切り替え不要
- **プロ仕様メディアプレイヤー** — 独自デュアルゾーンビデオシーク、アートワーク表示音楽プレイヤー
- **クロスプラットフォーム転送** — LAN ピアツーピア高速転送。Apple デバイス間は Wi-Fi 不要
- **生体認証プライベートスペース** — Face ID / Touch ID 保護。100% ローカル

## 🛡️ プライバシー

- ❌ アカウント登録なし
- ❌ 個人データ収集なし
