# PouchVerse（储物袋）

> **微に構え、簡に行う · すべてのファイルを一つに、瞬時に閲覧**  
> 散らばったファイルを統一ライブラリに集約し、検索・プレビュー・転送をシームレスに。

PouchVerse は**ローカルファースト / オフライン対応**の個人ファイル管理アプリです。アカウント登録不要、クラウドアップロードなし、サブスクなし。すべてのファイルは**お使いのデバイス内のみ**に保存されます。

---

## 万文帰一

### ファイルは一か所に収まったことがない

現実のファイルは常に散らばっています。

スマートフォンのアルバム、チャット履歴、ダウンロードフォルダ、各アプリにそれぞれ一つ。\
資料を探すたびに、場所を行ったり来たりし、何度も重複して保存することになります。

最初のアイデアはシンプルでした：\
すべてのファイルを一か所に集め、統一して管理する。

「ファイルを探す」という行為が、もはや運任せにならないように。

### 一枚の写真が生んだ変化

2026年夏のビーチチームビルディングで撮った写真があります。

従来の方法で保存しようとすると、どこかのフォルダに入れなければなりません：\
「2026」？「チームビルディング」？「海辺」？「仕事の写真」？

でもその写真自体は、一か所だけに属するものではありません。

そこで私たちは「保存する」ということを考え直し始めました。\
ファイルはもはや特定のフォルダに属する必要はなく、最初からパスに縛られません。

物理ストレージは隠され、ユーザーは直接構造と向き合わなくなります。\
代わりに情報そのものが解放され、複数の次元で同時に整理できます：\
2026、夏、海辺、チームビルディング——どれもその写真へのエントリーポイントになれます。

### ファイルは入った瞬間に理解される

ファイルがシステムにインポートされた時点で、すでに変化が起きています。

それはもはや静的なファイルではなく、再処理された情報ユニットになります：\
フィンガープリントが計算され、コンテンツが認識され、テキストが抽出され、OCRが完了し、インデックスが構築され、サムネイルが生成され、中国語テキストにはピンインと頭文字処理が行われます。

これらすべてが入った瞬間に起きます。

従来のフォルダはこれをしません。\
フォルダは「入れること」だけを担当します——ここでは「見つけられるようにすること」を担当します。

ファイルは入った瞬間から、理解可能で、検索可能で、関連付け可能になります。

### 情報はすべてのデバイス間を流れる

情報が構造化されると、それはデバイスに縛られなくなります。

スマートフォンとタブレットがコアノードになります：デジタル資産のインポート・管理・保持を担当。\
デスクトップが処理ノードになります：重要なコンテンツの整理・加工を行い、システムに還流させます。\
テレビが消費ノードになります：すべての情報が瞬時に同期され、ストリームとして閲覧・使用できます。

それぞれが独自のファイルシステムを維持するのではなく、同一の情報構造を共有します。

ファイルはもはや「どのデバイスにある」ではなく、常に同じシステムに属し、異なる場面を流れます。

## ✨ なぜ PouchVerse なのか？

スマートフォン上のファイルはメール・SNS・ブラウザに散乱しています。従来のファイルマネージャーは、デスクトップの「フォルダツリー」をそのまま移植したものが多く、このような断片化に対応できません。

**PouchVerse は新しいアプローチを取ります：**

- **万文帰一** — あらゆるアプリからファイルを集約し、統一インデックス管理
- **物理レベル重複排除** — SHA-256 フィンガープリントでコンテンツが同一なら 1 つの実体に統合
- **6 次元フレキシブル整理** — タグ・重要度・仮想フォルダ・用途ラベルで自由分類
- **ミリ秒オフライン検索** — ファイル名・本文・画像 OCR・音声文字起こし（手動トリガー）
- **全形式プレビュー** — 80 種以上対応。外部アプリへの切り替え不要
- **プロ仕様メディアプレイヤー** — 独自デュアルゾーンビデオシーク、アートワーク表示音楽プレイヤー
- **クロスプラットフォーム転送** — LAN ピアツーピア高速転送。Apple デバイス間は Wi-Fi 不要
- **生体認証プライベートスペース** — Face ID / Touch ID 保護。100% ローカル
