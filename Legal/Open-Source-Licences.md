# Open Source Licences

PouchVerse statically links the following open-source libraries. No data is transmitted through any of them.

| Library | Version | Licence | Use |
|---|---|---|---|
| [FFmpeg](https://ffmpeg.org/) | 6.x | LGPL 2.1+ | Local video / audio decoding |
| [unrar](https://www.rarlab.com/rar_add.htm) | 6.x | UnRAR Licence (extract-only) | RAR archive extraction |
| [zlib](https://zlib.net/) | System | zlib Licence | DEFLATE decompression |
| [SQLite3](https://sqlite.org/) | System | Public Domain | Local database (with FTS5) |
| [FreePats General MIDI soundfont](https://freepats.zenvoid.org/) | — | CC0 1.0 Universal | MIDI playback |

Full licence texts are available in-app under **Sidebar → Licences**.
