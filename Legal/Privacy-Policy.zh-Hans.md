# 储物袋 隐私政策

**生效日期：2026 年 4 月 26 日**
**适用产品：储物袋（PouchVerse）iOS 版**
**发布主体：北京艺简单网络科技有限公司（Beijing EJIANDAN Network Technology Co., Ltd.）**

---

## 一、概述

我们坚持"**本地优先 / 离线可用**"的产品原则。储物袋是一款本地文件聚合管理工具，
不要求注册、不收集账号、不上传任何文件或元数据到服务器。本政策旨在向您透明地说明：
本应用在您的 iOS 设备上**做什么**、**不做什么**，以及您可以**如何控制**。

---

## 二、我们**不**收集的信息

我们**不会**收集、存储、上传或分享下列任何信息：

- 您的姓名、电话、邮箱、身份证号或其它真实身份信息
- 您的地理位置、设备唯一标识（IDFA / IDFV）、广告标识
- 您导入、查看、组织、传输的任何文件内容或文件名
- 您的搜索关键词、浏览记录、使用行为
- 您的通讯录、日历、相册元数据、健康数据
- 任何用于跟踪、画像、营销、广告投放的数据

本应用**不内置任何第三方 SDK / 分析 / 广告 / 推送服务**。

---

## 三、本应用在设备上处理的数据

为实现核心功能，下列数据保存在**您设备的本应用沙盒内**，不会离开您的设备：

| 数据 | 存放位置 | 用途 |
|---|---|---|
| 您导入的文件本体 | `Documents/store/` | 提供管理与预览 |
| 文件元数据（名称、大小、类型、时间、颜色、用途等） | `Documents/pouchverse.db` | 列表、筛选、排序 |
| 全文索引（FTS5） | `Documents/pouchverse.db` | 关键词搜索 |
| 中文拼音 / 首字母、图片 OCR 文本与场景标签、音频语音转写文本 | `Documents/pouchverse.db`（全文索引内） | 增强本地检索；生成过程全部在本机离线完成 |
| 缩略图 | `Documents/thumbs/` | 列表预览 |
| 哈希值（SHA-256） | `Documents/pouchverse.db` | 内容去重 |
| 偏好设置 | `UserDefaults` | 视频跳转步长、局域网接收开关等 |

卸载本应用即等同于**完全删除**上述所有数据。

## 四、需要您授权的系统权限

| 权限 | 是否必需 | 用途 | 拒绝后的影响 |
|---|---|---|---|
| **文件 / 文档选择器** | 否（按需） | 通过 iOS"文件" App / 文档选择器导入文件 | 仅影响该来源的导入 |
| **照片库（如使用）** | 否 | 从相册导入图片或视频 | 仅影响相册导入 |
| **本地网络** | 否 | 局域网设备发现与文件透传 | 仅影响局域网传输功能 |
| **语音识别（Speech Recognition）** | 否 | 仅在用户在音频预览页主动点击「文字提取」时调用，强制「仅本机离线识别」，音频不出本机；导入时不会调用 | 仅影响音频文字提取功能；导入、预览与其他检索不受影响 |
| **Face ID / Touch ID / 密码** | 否 | 解锁应用内"私密文件"分区 | 仅影响私密文件访问 |

本应用**不申请**联网、推送、麦克风（仅语音识别能力调用 Speech 框架，不采集麦克风输入）、相机、定位、通讯录、日历、运动与健身等权限。

## 五、网络行为

- **不连接互联网**：本应用不会向任何远端服务器发送或接收数据。
- **本机离线智能索引**：拼音转写、图片 OCR / 场景分类、音频语音转写均调用苹果 **Vision** / **Speech** 系统框架在本机完成，全程不联网；其中语音识别强制 `requiresOnDeviceRecognition = true`。
- **局域网透传**：仅当您主动使用"局域网传输"功能时，本应用会通过 Bonjour（mDNS）发现同一局域网内的对端设备，并通过自定义 TCP 协议（端口 52013）直接传输您选择的文件。
- 局域网传输**不经过任何中转服务器**，发送方与接收方点对点直连；
  传输不加密，请勿在不可信的公共 Wi-Fi 网络下传输敏感数据。

## 六、私密文件

"私密文件"功能通过 iOS 系统的 Local Authentication（Face ID / Touch ID / 设备密码）
提供**应用内的二级访问控制**，用于隐藏私密文件夹中的内容。请注意：

- 私密文件**未在磁盘上加密**；其隐私边界依赖"成员关系标记 + 内存解锁态 + 界面隐藏"。
- 应用切换至后台或退出后**立即重锁**，重新打开需再次身份验证。
- 越狱设备、被他人物理获取并已知设备密码的设备，存在文件被读取的风险。

## 七、未成年人保护

本应用面向通用用户群体，不针对未成年人提供专门服务。如未满 14 周岁的未成年人使用，
请在监护人指导下使用，并由监护人协助管理设备权限与导入内容。

---

## 八、第三方组件

本应用静态集成了以下开源组件，**不通过它们传输任何数据**：

- **FFmpeg**（LGPL 2.1+）：本地视频/音频解码
- **unrar**（UnRAR License，仅解压）：RAR 解压
- **zlib**（zlib license）：DEFLATE 解压
- **SQLite3**（公有领域）：本地数据库

详细许可证文本可在 App 内"侧边栏 → 许可"查看。

---

## 九、数据导出与删除

- **导出**：通过列表行长按 → 分享，或直接将文件分享至"文件"App / 其他 App。
- **删除**：列表行左滑删除，或在 iOS 系统设置中卸载本 App，将彻底移除所有本地数据。
- 由于数据完全位于本机，**我们无法、也不会**远程恢复您已删除的数据。
