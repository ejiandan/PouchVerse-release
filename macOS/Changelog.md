# PouchVerse Transfer — macOS Changelog

---

## v1.0

**Status:** Submitted to Mac App Store — Launching June 16, 2026  
**TestFlight:** <https://testflight.apple.com/join/5Bk89gBb>

### Initial Release

#### High-Speed LAN Transfer

- Bonjour peer discovery + custom TCP direct transfer; no relay server, no account required.
- Cross-platform transfer between PouchVerse iOS and PouchVerse Transfer for Windows.
- Apple devices can transfer directly without Wi-Fi.

#### Simple Operation

- Clean, focused interface centered on file transfer.
- Supports batch file sending and receiving.
- Real-time transfer progress with cancel support.

#### Privacy & Security

- All transfers are peer-to-peer within the local network — files never pass through any server.
- No account registration, no cloud storage, no user data collection.

#### Multi-Language Support

- Simplified Chinese, Traditional Chinese, English, Japanese, Korean, Spanish.
